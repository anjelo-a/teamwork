# TeamWork

A real-time collaboration platform monorepo.

## Structure
- `apps/web` — frontend
- `apps/api` — backend and realtime server
- `packages/types` — shared types
- `packages/validation` — shared validation
- `AGENTS.md` — durable project guidance for AI-assisted development
- `PLANS.md` — planning template for complex work

## Security
Do not commit `.env` files or real credentials. Use `.env.example` as the template and keep actual secrets local or in your deployment platform's secret store.
